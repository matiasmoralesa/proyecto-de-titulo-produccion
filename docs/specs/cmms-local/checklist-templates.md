# Plantillas de Checklists - Sistema CMMS

Este documento contiene las 5 plantillas predefinidas de checklists para los diferentes tipos de vehículos del sistema CMMS.

## Información General

- **Total de Plantillas:** 5
- **Tipos de Vehículos:** Camión Supersucker, Camioneta MDO, Retroexcavadora MDO, Cargador Frontal MDO, Minicargador MDO
- **Formato:** Cada plantilla incluye secciones, preguntas, tipo de respuesta y si es requerida
- **Puntuación Mínima:** 80% para aprobar

## Tipos de Respuesta

- `yes_no_na`: Sí / No / No Aplica
- `text`: Texto libre
- `numeric`: Valor numérico
- `photo`: Requiere foto

## Notas de Implementación

### Estructura de Datos

Cada plantilla debe almacenarse en la base de datos con la siguiente estructura:

```python
{
    "code": "F-PR-020-CH01",
    "name": "Check List Camionetas MDO",
    "vehicle_type": "CAMIONETA_MDO",
    "description": "Checklist diario para Camionetas MDO",
    "is_system_template": True,  # No se puede modificar ni eliminar
    "passing_score": 80,
    "items": [
        {
            "section": "I - Auto Evaluación del Operador",
            "order": 1,
            "question": "Cumplo con descanso suficiente...",
            "response_type": "yes_no_na",
            "required": True,
            "observations_allowed": True
        },
        # ... más items
    ]
}
```

### Tipos de Respuesta

- **yes_no_na**: Respuesta Sí/No/No Aplica (más común)
- **text**: Texto libre para observaciones
- **numeric**: Valor numérico (ej: presión de neumáticos)
- **photo**: Requiere adjuntar foto

### Cálculo de Puntuación

```python
# Fórmula de puntuación
total_items = len(items)
items_passed = count(items where response == "yes" or response == "na")
score = (items_passed / total_items) * 100

# Aprobado si score >= passing_score (80%)
```

### Generación de PDF

El PDF generado debe:
1. Incluir encabezado con logo y código de plantilla
2. Mostrar información del vehículo y operador
3. Listar todas las secciones e items con respuestas
4. Incluir fotos adjuntas (si las hay)
5. Mostrar puntuación final y estado (Aprobado/Rechazado)
6. Incluir firma digital del operador
7. Fecha y hora de completado

### Comando de Carga

```bash
# Cargar plantillas en la base de datos
python manage.py load_checklist_templates
```

Este comando debe:
1. Leer las plantillas desde un archivo JSON
2. Crear o actualizar las plantillas en la base de datos
3. Marcar `is_system_template=True` para protegerlas
4. Validar la estructura de datos
